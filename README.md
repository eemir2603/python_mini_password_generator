 # Secure Password Generator

# Overview
This project marks **Day 1** of my intensive Python and Cybersecurity learning journey. As a foundational step into scripting and automation, this mini-project is a simple yet highly effective script designed to generate a random, complex 12-character password. 

# Features
* Built entirely with Python's built-in `random` module.
* Generates strong passwords utilizing a mix of uppercase letters, lowercase letters, integers, and special characters.
* Lightweight, fast, and optimized.
